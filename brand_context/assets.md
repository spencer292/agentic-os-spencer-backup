## Color Palette

| Name | Role | HEX | RGB | Usage |
|------|------|-----|-----|-------|
| **Grass** | Primary | #184241 | 24, 66, 65 | ~60% of design space. Main background color. |
| **Blue** | Secondary | #182034 | 24, 32, 52 | ~30% of design space. |
| **Cream** | Neutral | #FFF1D9 | 255, 241, 217 | Text on dark backgrounds, neutral backgrounds. |
| **Gold** | Accent | #E68C04 | 230, 140, 4 | ~10% accent. CTAs, highlights. |
| **Rust** | Accent | #8F2A2D | 143, 42, 45 | ~10% accent. Alerts, emphasis. |

### Color Usage Rules
- **Grass background:** Use Cream or Gold text/elements
- **Cream background:** Any color can be used
- **Blue background:** Use Cream or Gold text/elements
- **Gold background:** Use Cream, Grass, or Blue text/elements
- **Rust background:** Use Cream or Gold text/elements

## Typography

| Level | Font | Weight | Usage |
|-------|------|--------|-------|
| **Heading** | Lexend (or Acumin Variable) | Bold | Headers, titles |
| **Subheading** | Zilla Slab | SemiBold | Subheadings |
| **Body** | Zilla Slab | Regular | Body copy |

### Typography Rules
- No all-caps for long passages
- No letter spacing modifications
- Consistent line spacing
- Text never placed too close to edges
- Consistent alignment within documents
- Heading, subheading, and body must be visually distinct sizes

## Logo Suite

### Primary Logo — "GOT MOLES?™"
- **Style:** Bold display type with distressed/earthy texture on the bottom edge (mole hill silhouette integrated into letterforms)
- **Usage:** Default logo. Websites, footers, stationery, signage.
- **Clear space:** Height of the logo on all sides minimum
- **Color variants:** Grass on light, Cream on dark, Gold on dark, Blue, Rust — all provided in JPG/PNG/SVG
- **Files:** `FILES/JPG/PRIMARY LOGO FILES-01 to 07`, same in PNG and SVG

### Secondary Logo — Stacked "GOT / MOLES?™"
- **Style:** Same treatment, two-line stacked layout
- **Usage:** When primary doesn't fit (smaller/taller spaces)
- **Clear space:** Half the height of the logo on all sides
- **Files:** `FILES/JPG/SECONDARY LOGO FILES-08 to 14`, same in PNG and SVG

### Logo Mark — Mole Hill Silhouette
- **Style:** Stylised mole hill / earth mound shape
- **Usage:** Favicons, social profile pictures, stamps, confined spaces
- **Clear space:** 0.5x height of mark
- **Files:** `FILES/JPG/LOGO MARK FILES-15 to 21`, same in PNG and SVG

### Logo Rules — DO NOT
- Alter colors
- Add text, elements, strokes, or shadows
- Place inside a holding shape
- Rotate, stretch, condense, or outline

## Live Design Reference — ScoreApp Quiz

**URL:** https://score.got-moles.com/
**Platform:** ScoreApp (limited design options)
**Purpose:** Lead gen quiz for Meta campaigns — captures emails via personalised Mole Risk Report

**Use as design reference for the main site because:**
- Already implements the new brand palette (Grass, Blue, Cream, Gold, Rust)
- Already uses Lexend headings + Zilla Slab body
- Demonstrates how the color palette works in a real layout
- Pain-first headline approach matches the voice profile
- Trust bar layout (stats in a row) is a pattern to replicate on the homepage
- Gold CTAs on Grass background = high contrast, high visibility
- Score tier system (Early Signs / Active Problem / Code Red) could inform How It Works or TMCP page design
- Dark background with cream text = the dominant visual feel of the new brand

**Design patterns to carry to the main site:**
- Full-width dark (Grass) sections with Cream text
- Gold accent buttons — "FIND MY RISK SCORE →" style CTAs
- Trust bar: stats in a horizontal row, Gold numbers, Cream labels
- Section transitions: Grass → Blue → Grass for visual rhythm
- Card-style testimonial carousel on dark background
- Urgency messaging in Rust or Gold highlight

**Notes for correction on the quiz itself:**
- "WA's #1 Mole Removal" — unsubstantiated, needs replacing
- "7 Yrs In Business" — now 9 years, needs updating
