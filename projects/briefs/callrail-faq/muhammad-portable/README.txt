GOT MOLES PHONE TRAINING — SETUP (5 minutes)
=============================================

Hi Muhammad — this bundle sets up an AI role-play coach that plays
realistic Got Moles callers so you can practice before taking real calls.
Every scenario is built from real recorded customer calls.

WHAT'S IN THIS FOLDER
---------------------
1. project-instructions.md ........ paste this into Claude (step 3 below)
2. 2026-07-20_muhammad-faq-training.pdf ... THE ANSWER KEY - read this first
3. 2026-07-20_roleplay-scenarios.pdf ...... the 15 scenarios you'll face
   (.md copies of both are included too - these are for uploading in step 4)

SETUP
-----
1. Go to claude.ai and sign in (Spencer will tell you which account).
2. Click "Projects" -> "Create Project" -> name it: Phone Training
3. In the project, open "Instructions" (in the right side panel), and
   copy-paste the ENTIRE contents of project-instructions.md into it.
4. In the same side panel under "Files", upload these two files:
      2026-07-20_muhammad-faq-training.md
      2026-07-20_roleplay-scenarios.md
5. Start a new chat inside the project and type:  let's train

HOW A SESSION WORKS
-------------------
- Claude asks which mode: drill / scenario / shift / exam.
  Start with "drill" (rapid-fire questions), then "scenario".
- The phone "rings" and Claude plays a caller. Answer exactly as you
  would on the real Got Moles line. Say "pause" anytime to stop a call.
- After each call you get a score out of 100 and coaching. 80+ = pass.
- At the end of each session Claude prints a scorecard block.
  COPY IT AND TEXT/EMAIL IT TO SPENCER - that's how progress is tracked.

WEEK 1 GOAL
-----------
Read the FAQ guide PDF cover to cover, run the rapid-fire drill daily,
and pass all five Tier 1 scenarios twice.

Questions -> Spencer.
