# Side Project — $100K from The Route + Route Ready

## Goal

**$100,000 cumulative gross revenue** from the two side-project assets combined, as fast as possible, with AI doing the heavy lifting and Spencer's hands-on time held to a short, explicit list. This is the "how far can nothing-but-AI go" experiment — run it clean so the result means something.

## Scoreboard

One number: cumulative gross across both assets, reported in every weekly digest against $100K.

Counts: Gumroad sales (kits, bundles, freebie upsells), book royalties (Amazon), audiobook revenue (all channels), any licensing revenue from the Route playbook.
Milestones: first dollar (clock running, day 2/45) → $1K → $10K → $25K → $100K. Log date each is hit.

**Spencer only (by choice or hard requirement):**
- Handwritten notes + mailing signed copies (outreach dossier exists)

**AI (everything else):**

- Cross-funnel: book ↔ kits each direction; /book click tracking already live
