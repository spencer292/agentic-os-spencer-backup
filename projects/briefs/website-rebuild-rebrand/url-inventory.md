- **Blog posts:** Preserve exactly as-is (clean slugs, good structure)
- **Core pages:** Map old → new (e.g., /our-process/ → /how-it-works/)

2. All `/{name}/` (plain city slugs) → consolidated city page

4. All `/{city}-mole-{service}` variants → consolidated city page
5. All `-2` duplicate pages → primary page
6. All `/service/{name}/` → consolidated service page
