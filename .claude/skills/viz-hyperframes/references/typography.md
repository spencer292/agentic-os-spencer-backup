# Typography

## Anti-Monoculture Rule

Reject fonts every LLM defaults to: Inter, Roboto, Poppins, Playfair Display, Syne.
Syne is an instant AI design tell.

## Pairing Rules

- Never pair two sans-serifs — cross boundaries: serif + sans, sans + mono
- One expressive font per composition — multiple interesting fonts create confusion
- Extreme weight contrast: minimum 300 vs 900 for video (not 400 vs 700)
- Tension must be meaningful — pairing should embody content contradictions

## Video Scale

- Body text: minimum 20px
- Headlines: 60px+
- Labels: 16px+
- Tracking: tighten to -0.03em to -0.05em on display sizes (video compression)

## Selection Strategy

1. Name the register (voice/tone of content)
2. Imagine the typeface as a physical object
3. Reject first instincts (likely training-data defaults)
4. Cross-check against category expectations

## Dark Background Adjustments

- Use lighter font weights (350 instead of 400)
- Increase line-height by 0.05-0.1
- White text slightly more transparent than full #fff

## OpenType Features

- `tabular-nums` for aligned digit columns
- `diagonal-fractions` for ratios
- `all-small-caps` for abbreviations

## Register Switching

Assign different fonts to different communicative modes — treat fonts as conversation
voices, not hierarchy levels. "Motion is typography" — a 0.1s entrance creates different
meaning than a 2s fade in identical typefaces.
