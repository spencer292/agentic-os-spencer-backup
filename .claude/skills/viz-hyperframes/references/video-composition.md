# Video Composition

Rules for composing video frames. design.md defines brand identity; this file defines how to
compose a video frame.

## Density

Scenes require 8-10 visual elements minimum across three layers:

1. **Background texture** — radial glow, oversized ghost type, colour panel, grain, or grid
   (never solid flat colour)
2. **Midground content** — the core message via cards, stats, or code blocks
3. **Foreground accents** — dividers, labels, data bars, metadata details

A beat with 3 elements looks empty. A beat with 8-10 feels alive.

## Colour Treatment

- Brand accents at 15-25% opacity for atmospheric effects, full saturation for focal points
- Light canvases need bolder borders (2px+ solid), stronger structural elements, subtle texture
- Neutrals should tint toward brand hue, not dead grey

## Typography Scale (Video vs Web)

| Element | Video | Web |
|---------|-------|-----|
| Headlines | 64-120px | 32-48px |
| Body text | 28-42px | 14-16px |
| Labels | 18-24px | 12px |
| Padding | 60-140px | 16-32px |

## Frame Composition

- Two focal points minimum — eye needs a travel path
- Hero text occupies 60-80% of frame width
- Edge anchoring (pin left/top or right/bottom) — avoid centred floating
- Split frames: data panels paired with content; top bars with full-width below
- Structural elements (rules, dividers, border panels) create eye paths and animate well

## Motion Requirements

- Every decorative element needs ambient motion (breathing, drifting, pulsing)
- "Subtle reads as static at 30fps" — make motion visible
- Active backgrounds: radial glows, oversized faded type, subtle panels, hairlines
- Pure solid #000 reads as "nothing loaded"
