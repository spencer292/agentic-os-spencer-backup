# TikTok Posting Guide

Actionable posting intelligence distilled from platform algorithm data. For full algorithm mechanics, see `.claude/skills/viral-short-form-video-master/references/tiktok-algorithm-2026.md`.

---

## Algorithm Signals (Ranked by Weight)

| Signal | Weight | What It Means for Posting |
|--------|--------|---------------------------|
| Watch Time | Highest | Caption must hook immediately — every scroll-past kills you |
| Completion Rate | Very High | Duration must match content depth — don't pad |
| Replays | High (5x weight) | Build loop potential into hook and ending |
| Shares | High | Create "send to a friend" moments |
| Comments | Medium-High | Ask questions, use controversial takes |
| Follows | Medium | Niche authority drives follow-through |
| Likes | Lower (2025 update) | Don't optimize for likes — shallow signal |

## Cold Start System

Every video is tested with 200-500 viewers first. The first 24 hours are critical — 68% of total views happen in this window. Posting timing and caption quality directly impact whether you pass the cold start.

| Stage | Audience | What Gets You to Next Stage |
|-------|----------|----------------------------|
| Test 1 | 200-500 | First 3 seconds retention |
| Test 2 | 1,000-5,000 | Completion rate vs length |
| Test 3 | 10,000-50,000 | Early engagement velocity |
| Test 4 | 100,000+ | Replay rate, shares |

## Caption Strategy

**Algorithm reads captions for categorization.** Your caption text directly determines which audience pool TikTok tests your video with.

### Rules
1. Caption must clearly signal your niche — the algorithm uses it for micro-categorization
2. Include the core topic keyword in the first sentence
3. Keep it punchy: 1-2 sentences max
4. End with a CTA that drives comments or follows

### Hashtag Strategy

| Type | Count | Purpose |
|------|-------|---------|
| Micro-niche | 2-3 | Primary discovery (2.5x better distribution vs broad) |
| Category | 1-2 | Topic-level reach |
| Broad/trending | 0-1 | Only if genuinely relevant |

**Total: 3-5 hashtags.** Quality over quantity. Trending hashtags increase engagement 23% when relevant.

**Micro-niche examples:**
- Instead of `#fitness` → `#beginnerlworkoutsforwomen`
- Instead of `#tech` → `#aiautomationtools`
- Instead of `#business` → `#saasfounderlife`

## Loop & Replay Optimization

Replays carry the highest algorithm weight on TikTok. Optimize for them:

1. **Caption hook:** Tease a payoff that makes viewers want to rewatch ("Watch till the end" is weak — be specific)
2. **First comment:** Post immediately after publishing with an engagement question
3. **End connects to start:** If the video loops well, mention it in the caption

## Content the Algorithm Favors

- Family-friendly, positive emotional tone
- Educational value with clear takeaways
- Authentic creator content (not overproduced)
- Micro-niche targeting over broad appeal
- Trending sound usage (when relevant)

## Content the Algorithm Deprioritizes

- Watermarked content from other platforms
- Duplicated/reposted content
- Misleading thumbnails or captions
- Low-quality production
- Content that triggers safety systems

## Posting Schedule

| Metric | Value |
|--------|-------|
| Optimal frequency | 1-4x daily |
| Critical window | First 24 hours (68% of views) |
| Peak timing improvement | 30% (test your audience) |
| Growth boost at 2-4x daily | 2.5x follower growth |

**Key insight:** Volume matters on TikTok. More posts = more cold start tests = more chances to break through.

## Duration Sweet Spots

| Length | Best For |
|--------|----------|
| 15-30s | Peak viral performance — highest completion rates |
| Under 15s | Quick hooks, but less watch time signal |
| 30-60s | Educational content, moderate completion |
| 60s+ | Only if content sustains attention throughout |

**For pipeline clips:** If `target_platform: tiktok`, aim for 15-35s clips.
