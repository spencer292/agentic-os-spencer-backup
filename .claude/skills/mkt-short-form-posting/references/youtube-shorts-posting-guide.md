- Algorithm measures "Viewed vs. Swiped Away" ratio

## Title Strategy

YouTube Shorts titles are critical — they function as both discovery mechanism and hook reinforcement.

### Rules
1. **Max 100 characters** (shorter is better for mobile)
2. **Front-load the keyword** — algorithm and viewers scan left-to-right
3. **Include #Shorts** for discovery (still matters in 2026)
4. **Power words:** Free, Proven, Easy, Fast, Secret, Hidden
5. **Keyword-rich** — think search intent, not just clever phrasing

### Title Patterns That Work
- `[Stat] — Here's Why #Shorts` (curiosity)
- `Stop [Bad Thing] — Do This Instead #Shorts` (contrarian)
- `How [Outcome] in [Timeframe] #Shorts` (how-to)
- `[Number] [Things] You Didn't Know About [Topic] #Shorts` (listicle)

### First Frame as Thumbnail
In some feed views, the first frame acts as the thumbnail. Consider this when the video opens — the first frame should be visually compelling even as a still image.

## CTA Strategy

YouTube Shorts with strong CTAs see **22% more engagement**. Always include one.

| CTA Type | When to Use |
|----------|-------------|
| "Full tutorial on channel" | Pipeline clips from long-form (41% faster growth with Shorts + long-form) |
| "Subscribe for more" | Channel growth focus |
| "Comment your answer" | Engagement boost |
| "Save this for later" | Save signal |

**Shorts + Long-Form Synergy:** Channels using both formats see 41% faster growth. Always tease the full video when clips come from long-form content.

- Consistent posting maintains momentum
- Old Shorts rarely resurface
- Volume + consistency > sporadic quality

## Duration Sweet Spot

| Length | Views vs 15s Baseline |
|--------|----------------------|
| 15s | 1x (baseline) |
| 30s | 1.5x |
| 45s | 2.2x |
| **55s** | **3x (sweet spot)** |
| 60s | 2.8x |

**Why 55 seconds:** Enough time to deliver value, long enough for pattern interrupts, still achieves high completion rates.

**For pipeline clips:** If `target_platform: youtube`, aim for 45-60s clips.
