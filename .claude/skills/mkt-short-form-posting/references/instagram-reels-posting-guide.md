## December 2025 Major Updates

### "Your Algorithm" Feature
Users can now see and customize what topics Instagram thinks they like. This means:
- **Topic clarity is critical** — mixed-topic creators see reduced reach
- Algorithm categorizes you based on consistent content themes
- Niche focus = significantly better distribution

### Trial Reels
Test content with non-followers without risking engagement metrics:
1. Create a Reel → select "Trial Reel"
2. Shown only to non-followers
3. Good performance → publish to feed
4. Poor performance → delete without impact

**When to recommend:** New content formats, different hooks, topic experiments.

## The 3/8/12 Rule

Instagram evaluates Reels at three algorithm checkpoints:

| Checkpoint | Time | What Happens |
|------------|------|-------------|
| **Hook** | 3 seconds | Continue or abandon decision |
| **Deepen** | 8 seconds | Interest prediction — is this worth showing more? |
| **Deliver** | 12+ seconds | Distribution decision — push to Explore or not |

**Caption implication:** The first line of your caption must reinforce the 3-second hook. If someone reads the caption first (common on IG), it should make them want to watch.

## Sends Per Reach (The #1 Signal)

Content that gets DM-shared tells Instagram "this is worth showing to more people." This is THE most powerful distribution signal.

### How to Optimize Captions for Sends
1. **Frame content as "send-worthy"** — "Tag someone who needs this" or "Send this to your [role]"
2. Create relatable content people want to share privately
3. Include specific, useful information worth forwarding
4. Spark conversations between friends

### Caption Framing Examples
- Instead of: "Here's how to grow on Instagram" → "Your friend who's been struggling with reach needs to see this"
- Instead of: "5 AI tools for marketers" → "Send this to someone still doing this manually"

## Caption Strategy

### The Shift: Hashtags to Keywords (2025-2026)

Instagram is moving from hashtag-based to **keyword-based discovery**. Captions are now searchable.

**Old Strategy (Outdated):**
- 30 hashtags per post
- Mix of popular and niche
- Hashtags in comments

**New Strategy (2026):**
- 3-5 highly relevant hashtags only
- Focus on **keywords in caption text**
- Write searchable, descriptive captions
- Think "what would someone search for?"

**Stats:**
- Posts with at least one hashtag: 12.6% more engagement
- Excessive hashtags: diminishing returns
- Keyword-rich captions: improved discovery

### Caption Format
```
[One punchy sentence about the problem/solution — reinforces the 3s hook]
[Optional second sentence with the key insight — aligns with 8s checkpoint]
[CTA — "Send this to someone who..." or "Save this for later"]

#keyword1 #keyword2 #keyword3
```

**Keep it minimal:** 1-3 sentences maximum. The caption should be readable in 2 seconds.

### Hashtag Rules
- **3-5 highly relevant hashtags only**
- Keyword-relevant, not vanity metrics
- No generic hashtags (#instagood, #photooftheday)
- Think searchable topic keywords

## First Comment Strategy

Post immediately after publishing:
```
"Save this and share it with someone who needs to hear it."
```

The first comment drives early engagement velocity, which affects the 3/8/12 evaluation window.

## 50% Watch Without Sound

Per Adam Mosseri, 50% of Instagram users watch without sound. This means:
- Captions on the video are essential (our pipeline handles this)
- Caption text in the post description should stand alone
- Key information must be visually conveyed, not just spoken

## Optimal Video Length

| Length | Use Case | Performance |
|--------|----------|-------------|
| 7-15s | Quick tips, reactions | Highest completion rates |
| 15-30s | Most content | Viral sweet spot |
| 30-60s | Tutorials, stories | Optimal engagement |
| 60-90s | Deep dives | Best for retention |

**Key insight:** 7-30s for viral potential, 30-90s for optimal engagement.

**For pipeline clips:** If `target_platform: instagram`, aim for 30-60s clips.
