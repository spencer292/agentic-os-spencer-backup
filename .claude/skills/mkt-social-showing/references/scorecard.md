# Pre-Post Scorecard

Score every package before it posts. Show the user the result with the package.

Rate 1-5 on each dimension:

| # | Dimension | 5 = | 1 = |
|---|-----------|-----|-----|
| 1 | **Hook strength** | On-screen text + first line stop the scroll in 3s, stack 2+ triggers | Slow build, throat-clearing, no clear stop |
| 2 | **ICP fit** | Clearly speaks to one persona's pain/aspiration; they'd think "this is me" | Generic; no line from content to their life |
| 3 | **Shareability** | An obvious identity/usefulness/emotion reason to save or send | Forgettable; no reason to pass on |
| 4 | **Clarity** | One message, instantly gettable | Two+ ideas, confusion |
| 5 | **Platform fit** | Native per platform (LinkedIn link-in-comment, keywords for search, right length) | Same caption pasted everywhere |

## Gate

- **Hook or ICP fit below 4 → revise, do not post.** These two are non-negotiable.
- Total below 18/25 → revise.
- 18-21 → ship, note the weak dimension.
- 22-25 → ship with confidence.

## Output format (in `showing-brief.md`)

```
SHOWING SCORE — {job}
Hook 4/5 · ICP fit 5/5 · Shareability 4/5 · Clarity 5/5 · Platform fit 4/5 = 22/25 ✅
Hook: "{on-screen}" / "{first line}"  (triggers: self-relevance + emotion)
ICP angle: {one line — which persona, which pain/aspiration}
Weak spot: {what to improve, if any}
```
