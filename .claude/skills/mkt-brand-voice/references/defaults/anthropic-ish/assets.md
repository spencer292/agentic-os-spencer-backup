# Default Brand — Visual Assets

A neutral, editorial-leaning default. Off-white paper background, charcoal ink, chartreuse accent. Inspired by the Anthropic Design System aesthetic — refined, technical, confident.

Use this when the user has no brand context yet but wants to ship. Encourages them to override piece-by-piece.

## Colors

- **Primary:** `#14130f` — charcoal ink. Headlines, primary buttons, dominant text.
- **Secondary:** `#6c6962` — warm gray. Body copy on light surfaces, dividers.
- **Background:** `#f7f5f0` — warm off-white paper. Slide canvas, content surfaces.
- **Text:** `#14130f` — same as primary on light, inverts to `#f7f5f0` on dark surfaces.

Optional additional palette colors:
- **Accent 1:** `#c4d600` — chartreuse. CTAs, highlights, sketch annotations.
- **Accent 2:** `#c4543b` — terracotta. Italic emphasis, "needs work" signals, warm accents.

## Sketch Style

- **Sketch color:** `#c4543b` — terracotta for hand-drawn emphasis on screenshots
- **Sketch weight:** medium
- **Sketch personality:** loose — intentional imperfection, slight hand wobble
