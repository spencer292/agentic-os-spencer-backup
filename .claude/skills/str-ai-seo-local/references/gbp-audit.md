# Google Business Profile Audit — Per Location

Got Moles has 3 GBPs (Seattle, Tacoma, Enumclaw). Run this checklist for each. 2026 ranking weights: proximity ~55% (uncontrollable), GBP signals 32%, reviews 16-20%, on-page 19%. Primary category is the single highest controllable factor.

## Per-location scorecard (max 100)

| Section | Max | What "pass" looks like in 2026 |
|---------|:-:|---|
| Primary category | 10 | "Pest Control Service" exactly. Wrong primary category = single biggest unforced error. |
| Secondary categories | 5 | Up to 9 used. Suggested: Mole Trapper, Wildlife Control Service, Lawn Care Service, Animal Trapper, Animal Removal Service. Don't dilute with unrelated. |
| Business name | 5 | Real legal name only. No keyword stuffing (Google penalizes). "Got Moles? Mole Removal Seattle" = violation. |
| NAP consistency | 10 | Address + phone match site footer + sitemap structured data + Yelp + Apple Business Connect + Bing Places + BBB. One mismatch weakens entity trust. |
| Service area | 5 | Explicit city list (not radius) for SAB businesses. List the 20 served cities, not "20 mile radius." |
| Service list | 5 | Each service has name + description + price posture (starting at / quote-based). TMCP, One-Time Removal, Commercial. |
| Hours | 3 | Set, accurate, special hours for holidays. "Open now" surfaces in Map Pack. |
| Photos — count | 5 | 30+ minimum per location. Pro photos = +35% CTR (Semrush 2026). |
| Photos — types | 5 | Logo + cover + interior (truck/equipment/uniform) + team + before/after of mole work + service-in-progress. Geo-tagged where possible. |
| Photo freshness | 3 | At least one new photo in last 30 days. Stale photo libraries signal inactive listing. |
| Review count | 5 | Got Moles aggregate ≥219. Per-location target: Seattle 135 → 200+, Tacoma + Enumclaw similar. Mole Masters has 356 — review-volume gap is a documented launch lever. |
| Review velocity | 8 | New reviews in last 30 / 60 / 90 days. Velocity > total. Target: 8+ new per location per month. |
| Review response rate | 7 | 100% within 48 hours, including 5-stars (Google reads response density as engagement signal). |
| Review keyword density | 4 | Reviews naturally containing service keywords ("mole removal", "mounds", city name). Don't ask for them — but track presence. |
| Posts cadence | 5 | At least 1 Update / Offer / Event per week. Posts decay after 7 days for ranking purposes. |
| Q&A | 3 | All questions owner-answered, no orphans. Seed 5-8 likely questions per location with owner answers. |
| Attributes | 3 | Veteran-Owned ✓ (Spencer Hill, US Army 2011-2014), licensed/insured, online appointments, on-site services. |
| Messaging | 3 | On with response-time tracking, OR explicitly off if not staffed. Half-on (slow response) hurts ranking. |
| Booking link | 2 | Routes to score.got-moles.com quiz or Jobber direct, depending on funnel decision. |
| Products | 3 | Add TMCP / One-Time / Commercial as Products with pricing posture. Surfaces in Map Pack carousel. |
| Highlights / opening date | 1 | Opening date 2017 set. |

## Per-location red flags

- Same photos across all 3 locations → Google detects, downweights all 3
- NAP mismatch on any one citation → entity confusion across all 3
- Phone number forwards to a single number but listed differently → fine if the listed number redirects, but pick one display number per GBP
- Service area lists same cities in all 3 GBPs → competing with self for proximity weight; carve clean radii

## Cross-location coordination

3 GBPs sharing 1 brand have a unique risk: keyword cannibalization in the Map Pack. Mitigations:
- Distinct service-area lists per GBP (Seattle covers Eastside + city; Tacoma covers South Sound; Enumclaw covers Plateau + rural east)
- Distinct primary photos per location (don't share the same hero image)
- Each GBP's "From the business" description focuses on that area's specifics
- One canonical service page per service, but each city page on the site links to the closest GBP
